A professional agency landing page engineered to communicate technical authority and high-end capabilities. This project demonstrates advanced frontend layout techniques and modern animation principles using vanilla web technologies.

Key Features:

Scroll-Reveal Animations: Utilizes the IntersectionObserver API to smoothly trigger CSS animations as elements enter the viewport, optimizing performance by avoiding heavy scroll event listeners.

Immersive Aesthetic: Features a deep dark-mode palette with fixed radial background gradients and glassmorphism (backdrop-filter) effects on the sticky navigation bar.

Asynchronous Form Simulation: The contact form is wired with validation logic and a simulated network delay, transitioning the submit button into a spinning loader before displaying a success state—mimicking production-level API handling.

Responsive Architecture: Scales perfectly from desktop to mobile using fluid typography (clamp()) and CSS Grid.

Tech Stack: HTML5, CSS3 (Clamp/IntersectionObserver/Gradients), Vanilla JavaScript.
